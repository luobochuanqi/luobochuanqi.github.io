```cpp
// 错误写法
double myList[4];
```

* 因为在 C/C++ 中，数组名是一个常量指针，不能直接用花括号给它赋值（这种赋值方式只允许在声明数组的同时进行）

```c
x = y; // error, 不能直接将一个数组赋值给另一个数组，应该将数组元素一一对应赋值
```

```c
#include "string.h"
scanf("%s", name); // 无需&，name本身是数组首地址
```

```c
int b = 10;
// * 的不同用法
int* a; // 指针类型变量
*a = b; // 解引用
```

* 无 string 类，字符串必须用char[]或char*存储，搭配<string.h>的strlen/strcpy/strcmp等函数操作。

| 函数         | 功能                          | 核心注意事项                                  |
|--------------|-------------------------------|-----------------------------------------------|
| `strlen(s)`  | 求长度（不含`\0`）            | 依赖`\0`，无则结果错误                        |
| `strcmp(a,b)`| 比较字符串                    | 用`==0`判断相等，而非`a==b`（错误！）         |
| `strcpy(dst,src)` | 复制字符串                 | `dst`必须足够大，且可修改                     |
| `strcat(dst,src)` | 拼接字符串                 | `dst`剩余空间需足够，自动加`\0`               |
| `strncpy`/`strncat` | 安全版本复制/拼接       | `strncpy`可能需手动补`\0`，`strncat`自动补    |
